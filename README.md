# Project2_Data_Classification_AI

## Overview
This project demonstrates Data Classification using the Decision Tree Classifier on the Iris Dataset.

## Technologies Used
- Python
- Scikit-learn
- Google Colab

## Libraries
- load_iris
- train_test_split
- DecisionTreeClassifier
- accuracy_score

## Workflow
1. Load Iris Dataset
2. Split data into training and testing sets
3. Train the Decision Tree model
4. Predict flower classes
5. Evaluate model accuracy

## Output
- Accuracy: 1.0 (100%)
